# Problem I. 小v爱3 II

小v 非常喜欢数字“3”，现在在一个 $n \times m$ 的棋盘上玩游戏。每次小v 会在两个不同的空格子上各放一个棋子，这两个格子的曼哈顿距离恰好为 $3$。

曼哈顿距离定义为：$(x_1, y_1)$ 和 $(x_2, y_2)$ 之间的距离为 $|x_1 - x_2| + |y_1 - y_2|$。

小v 想知道，最多能在棋盘上放多少个棋子？

\InputFile
一行两个整数 $n, m$（$1 \leq n, m \leq 10^9$），表示棋盘的行数和列数。

\OutputFile
输出最多能放的棋子数。

\Examples
```
2 2
3 3
```

```
0
8
```
