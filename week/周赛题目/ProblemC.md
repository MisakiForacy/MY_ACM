# Problem C. 唯一数

Polycarp 有一个正整数 $x$，请你帮 Polycarp 构造一个所有数位互不相同且各位数字之和等于 $x$ 的最小正整数。如果不存在这样的数，输出 $-1$。

\InputFile
第一行一个整数 $t$（$1 \le t \le 50$），表示测试用例组数。

接下来 $t$ 行，每行一个整数 $x$（$1 \le x \le 50$）。

\OutputFile
每组测试输出一行，输出满足条件的最小正整数，若不存在则输出 $-1$。

\Examples
```
4
1
5
15
50
```

```
1
5
69
-1
```

\Note
在第三组测试用例中，满足条件的最小正整数是 $69$，因为其数位和为 $15$，且所有数位互不相同。
