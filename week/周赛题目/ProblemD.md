# Problem D

给定一个正整数 $x$。找到一个最小的正整数，其各位数字之和等于 $x$，并且所有的数字都是**互不相同**的。

\InputFile
第一行包含一个正整数 $t$（$1 \le t \le 50$）~--- 测试用例的数量。接下来是 $t$ 个测试用例。

每个测试用例包含一个正整数 $x$（$1 \le x \le 50$）。

\OutputFile
输出 $t$ 个测试用例的答案：

\begin{itemize}
\item 如果存在一个正整数，其各位数字之和等于 $x$，并且所有数字互不相同，输出这样的最小数字；
\item 否则输出 \t{-1}。
\end{itemize}
