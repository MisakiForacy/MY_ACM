# Problem A. 铲子与剑

Polycarp 玩一个著名的电脑游戏，在这个游戏中，他可以合成两种工具——铲子和剑。

合成一把铲子需要消耗 $2$ 根木棍和 $1$ 颗钻石；合成一把剑需要消耗 $2$ 颗钻石和 $1$ 根木棍。

每合成一件工具都可以卖出 $1$ 个绿宝石。现在 Polycarp 有 $a$ 根木棍和 $b$ 颗钻石，他最多能获得多少个绿宝石？

\InputFile
第一行一个整数 $t$（$1 \le t \le 1000$），表示测试用例组数。

接下来 $t$ 行，每行两个整数 $a$ 和 $b$（$0 \le a, b \le 10^9$），分别表示木棍和钻石的数量。

\OutputFile
每组测试输出一行，输出 Polycarp 最多能获得的绿宝石数量。

\Examples
```
4
4 4
1000000000 0
7 15
8 7
```

```
2
0
7
5
```

\Note
第一组：可以合成 $1$ 把剑和 $1$ 把铲子，共 $2$ 个绿宝石。
第二组：没有钻石，无法合成任何工具。
第三组：可以合成 $7$ 把工具。
第四组：可以合成 $5$ 把工具。
