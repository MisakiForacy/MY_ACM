# Problem F. k-树

给定一个长度为 $n$ 的数组 $a$，初始时数组中的元素均为 0。你需要对该数组进行 $n$ 次操作：在第 $i$ 次操作中，执行以下操作序列：

\begin{enumerate}
\item 从数组中选择一个由\textbf{仅包含 0 的最长子数组（连续子段）}，在所有这样的子段中选择\textbf{最左边}的一个；
\item 设该子段为 $[l; r]$。如果 $r-l+1$ 是奇数（不能被 $2$ 整除），则将 $a[\frac{l+r}{2}] := i$（其中 $i$ 是当前操作的编号）；否则（如果 $r-l+1$ 是偶数），则将 $a[\frac{l+r-1}{2}] := i$。
\end{enumerate}

考虑一个长度为 $5$ 的数组 $a$（初始时 $a=[0, 0, 0, 0, 0]$）。然后它的变化如下：

\begin{enumerate}
\item 首先，我们选择子段 $[1; 5]$ 并将 $a[3] := 1$，因此 $a$ 变为 $[0, 0, 1, 0, 0]$；
\item 然后我们选择子段 $[1; 2]$ 并将 $a[1] := 2$，因此 $a$ 变为 $[2, 0, 1, 0, 0]$；
\item 然后我们选择子段 $[4; 5]$ 并将 $a[4] := 3$，因此 $a$ 变为 $[2, 0, 1, 3, 0]$；
\item 然后我们选择子段 $[2; 2]$ 并将 $a[2] := 4$，因此 $a$ 变为 $[2, 4, 1, 3, 0]$；
\item 最后我们选择子段 $[5; 5]$ 并将 $a[5] := 5$，因此 $a$ 变为 $[2, 4, 1, 3, 5]$。
\end{enumerate}

你的任务是找到在执行完所有 $n$ 次操作后，长度为 $n$ 的数组 $a$。\textbf{注意答案是存在且唯一的}。

你需要回答 $t$ 个独立的测试用例。

\InputFile
输入的第一行包含一个整数 $t$（$1 \le t \le 10^4$）--- 测试用例的数量。接下来是 $t$ 个测试用例。

每个测试用例的唯一一行包含一个整数 $n$（$1 \le n \le 2 \cdot 10^5$）--- 数组 $a$ 的长度。

保证所有测试用例中 $n$ 的总和不超过 $2 \cdot 10^5$（$\sum n \le 2 \cdot 10^5$）。

\OutputFile
对于每个测试用例，输出答案 --- 长度为 $n$ 的数组 $a$，表示按照题目描述的操作后得到的结果。\textbf{注意答案是存在且唯一的}。
