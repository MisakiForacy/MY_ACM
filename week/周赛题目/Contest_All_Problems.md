# Problem A. 钉子与剑

**题目描述**

Polycarp 玩一个著名的电脑游戏（名字就不提了）。在这个游戏中，他可以合成两种工具——钉子和剑。

合成一把钉子需要消耗 2 根木棍和 1 颗钻石；合成一把剑需要消耗 2 颗钻石和 1 根木棍。

每合成一件工具都可以卖出 1 个绿宝石。现在 Polycarp 有 $a$ 根木棍和 $b$ 颗钻石，他最多能获得多少个绿宝石？

**输入格式**

第一行一个整数 $t$（$1 \leq t \leq 1000$），表示测试用例组数。
接下来 $t$ 行，每行两个整数 $a$ 和 $b$（$0 \leq a, b \leq 10^9$），分别表示木棍和钻石的数量。

**输出格式**

每组测试输出一行，输出 Polycarp 最多能获得的绿宝石数量。

**样例输入**

```
4
4 4
1000000000 0
7 15
8 7
```

**样例输出**

```
2
0
7
5
```

**样例说明**

第一组：可以合成 1 把剑和 1 把钉子，共 2 个绿宝石。
第二组：没有钻石，无法合成任何工具。
第三组：可以合成 7 把工具。
第四组：可以合成 5 把工具。

# Problem B. 移动括号

**题目描述**

给定一个长度为 $n$ 的括号序列 $s$，其中 $n$ 是偶数，$s$ 由 $n/2$ 个左括号 '(' 和 $n/2$ 个右括号 ')' 组成。

每次操作，你可以选择一个括号，将其移动到字符串的开头或结尾。

你的任务是，最少需要多少次操作，才能将 $s$ 变成一个合法的括号序列？

合法括号序列定义如下：

- "()" 是合法的；
- 如果 $A$ 是合法的，则 "(" + $A$ + ")" 也是合法的；
- 如果 $A$ 和 $B$ 都是合法的，则 $A$ + $B$ 也是合法的。

**输入格式**

第一行一个整数 $t$（$1 \leq t \leq 2000$），表示测试用例组数。
每组测试用例包含两行：
第一行一个偶数 $n$（$2 \leq n \leq 50$），第二行一个长度为 $n$ 的字符串 $s$，由 $n/2$ 个 '(' 和 $n/2$ 个 ')' 组成。

**输出格式**

每组测试输出一行，输出最少操作次数。

**样例输入**

```
4
2
)(
4
()()
8
())()()(
10
)))((((())
```

**样例输出**

```
1
0
1
3
```

**样例说明**

第一组：将第一个括号移动到末尾即可。
第三组：将最后一个括号移动到开头即可。
第四组：将最后三个左括号移动到开头即可。

# Problem C. 唯一数

**题目描述**

给定一个正整数 $x$，请你构造一个所有数位互不相同且各位数字之和等于 $x$ 的最小正整数。如果不存在这样的数，输出 $-1$。

**输入格式**

第一行一个整数 $t$（$1 \leq t \leq 50$），表示测试用例组数。
接下来 $t$ 行，每行一个整数 $x$（$1 \leq x \leq 50$）。

**输出格式**

每组测试输出一行，输出满足条件的最小正整数，若不存在则输出 $-1$。

**样例输入**

```
4
1
5
15
50
```

**样例输出**

```
1
5
69
-1
```

# Problem D. 第 k 个不是 n 的倍数的数

**题目描述**

给定两个正整数 $n$ 和 $k$，请你输出第 $k$ 个不是 $n$ 的倍数的正整数。

例如，$n=3$，$k=7$ 时，不是 $3$ 的倍数的正整数依次为 $1,2,4,5,7,8,10,11,13,\ldots$，第 $7$ 个是 $10$。

**输入格式**

第一行一个整数 $t$（$1 \leq t \leq 1000$），表示测试用例组数。
接下来 $t$ 行，每行两个整数 $n$ 和 $k$（$2 \leq n \leq 10^9$，$1 \leq k \leq 10^9$）。

**输出格式**

每组测试输出一行，输出第 $k$ 个不是 $n$ 的倍数的正整数。

**样例输入**

```
6
3 7
4 12
2 1000000000
7 97
1000000000 1000000000
2 1
```

**样例输出**

```
10
15
1999999999
113
1000000001
1
```

# Problem E. 构造数组

**题目描述**

你有一个长度为 $n$ 的数组 $a$，初始所有元素均为 $0$。你将对该数组进行 $n$ 次操作，第 $i$ 次操作如下：

1. 选择所有连续的 $0$ 段中最长且最靠左的一段，设其下标区间为 $[l, r]$。
2. 如果该段长度为奇数，则将中间位置 $a_{(l+r)/2}$ 赋值为 $i$；如果为偶数，则将最左的中间位置 $a_{(l+r-1)/2}$ 赋值为 $i$。
3. 重复上述过程，直到所有位置都被赋值。

请输出最终数组。

**输入格式**

第一行一个整数 $t$（$1 \leq t \leq 10^4$），表示测试用例组数。
接下来 $t$ 行，每行一个整数 $n$（$1 \leq n \leq 2 \times 10^5$）。保证所有 $n$ 之和不超过 $2 \times 10^5$。

**输出格式**

每组测试输出一行，输出最终数组。

**样例输入**

```
6
1
2
3
4
5
6
```

**样例输出**

```
1
1 2
2 1 3
3 1 2 4
2 4 1 3 5
3 4 1 5 2 6
```

# Problem F. k-树

**题目描述**

Lesha 设计了一种叫做 k-树 的无限根树：

- 每个节点有 $k$ 个子节点；
- 每条边有权值，且从某节点出发的 $k$ 条边权值分别为 $1,2,\ldots,k$。

Dima 想知道：从根节点出发，路径权值和恰好为 $n$，且路径上至少有一条边权值不小于 $d$ 的路径有多少条？答案对 $10^9+7$ 取模。

**输入格式**

一行三个整数 $n, k, d$（$1 \leq n, k \leq 100$，$1 \leq d \leq k$）。

**输出格式**

输出满足条件的路径条数，对 $10^9+7$ 取模。

**样例输入**

```
3 3 2
```

**样例输出**

```
3
```

**样例输入2**

```
3 3 3
```

**样例输出2**

```
1
```

# Problem G. Phoenix 的分配

**题目描述**

Phoenix 有一个只包含小写字母的字符串 $s$，他想把所有字母分成 $k$ 个非空字符串 $t_1, t_2, \ldots, t_k$，每个字母只能属于一个 $t_i$。每个 $t_i$ 内部可以任意重排。

Phoenix 想让 $k$ 个字符串中字典序最大的字符串尽量小。请你输出最小可能的最大字典序字符串。

**输入格式**

第一行两个整数 $n, k$（$1 \leq k \leq n \leq 10^5$），第二行一个长度为 $n$ 的字符串 $s$。
保证所有 $n$ 之和不超过 $10^5$。

**输出格式**

每组测试输出一行，输出最小可能的最大字典序字符串。

**样例输入**

```
6
4 2
baba
5 2
baacb
5 3
baacb
5 3
aaaaa
6 4
aaxxzz
7 1
phoenix
```

**样例输出**

```
ab
abbc
b
aa
x
nehinopx
```

# Problem H. Linova 和王国

**题目描述**

Linova 梦见了一个神奇的王国，这个王国有 $n$ 个城市和 $n-1$ 条双向道路，城市编号为 $1$ 到 $n$，城市 $1$ 是首都。任意两个城市之间都可以通过道路互达（即这是一棵树）。

Linova 计划选择恰好 $k$ 个城市发展工业，其余城市发展旅游。首都也可以被选为工业或旅游城市。

每年在首都召开一次会议，每个工业城市会派一名使者前往首都。每位使者会选择一条最短路径前往首都（且唯一）。

每位使者在旅游城市中旅行会感到愉快。对于每位使者，他的幸福值等于他路径上经过的旅游城市数量。

Linova 想选择 $k$ 个城市发展工业，使所有使者的幸福值之和最大。请你帮她计算最大可能的幸福值总和。

**输入格式**

第一行两个整数 $n, k$（$2 \leq n \leq 2 \times 10^5$，$1 \leq k < n$）。
接下来 $n-1$ 行，每行两个整数 $u, v$，表示 $u$ 和 $v$ 之间有一条道路。
保证任意两个城市之间都连通。

**输出格式**

输出最大幸福值总和。

**样例输入**

```
7 4
1 2
1 3
1 4
3 5
3 6
4 7
```

**样例输出**

```
7
```

# Problem I. 小C爱3 II

**题目描述**

小C非常喜欢数字“3”，他现在在一个 $n \times m$ 的棋盘上玩游戏。每次他会在两个不同的空格子上各放一个棋子，这两个格子的曼哈顿距离恰好为 $3$。

曼哈顿距离定义为：$(x_1, y_1)$ 和 $(x_2, y_2)$ 之间的距离为 $|x_1 - x_2| + |y_1 - y_2|$。

他想知道，最多能在棋盘上放多少个棋子？

**输入格式**

一行两个整数 $n, m$（$1 \leq n, m \leq 10^9$），表示棋盘的行数和列数。

**输出格式**

输出最多能放的棋子数。

**样例输入**

```
2 2
3 3
```

**样例输出**

```
0
8
```

# Problem J. 排列的平方根

**题目描述**

给定一个长度为 $n$ 的排列 $p$（即 $1$ 到 $n$ 的一个排列）。

对于一个排列 $q$，定义 $q^2$ 为 $p$，即 $p[i] = q[q[i]]$ 对所有 $i$ 都成立。

现在给定 $p$，请你求出任意一个满足条件的排列 $q$，如果不存在，输出 $-1$。

**输入格式**

第一行一个整数 $n$（$1 \leq n \leq 10^6$）。
第二行 $n$ 个整数 $p_1, p_2, \ldots, p_n$，为 $p$ 的排列。

**输出格式**

如果存在，输出一行 $n$ 个整数 $q_1, q_2, \ldots, q_n$，否则输出 $-1$。

**样例输入**

```
4
2 1 4 3
```

**样例输出**

```
3 4 2 1
```

# Problem K. Nastya 和游戏

**题目描述**

Nastya 得到一个长度为 $n$ 的数组 $a$，她想玩一个游戏：选择一个子段，使得该子段所有元素的乘积等于其和乘以 $k$。

即对于子段 $a_l, a_{l+1}, \ldots, a_r$，满足 $a_l \times a_{l+1} \times \cdots \times a_r = (a_l + a_{l+1} + \cdots + a_r) \times k$。

请你计算有多少个子段满足上述条件。

**输入格式**

第一行两个整数 $n, k$（$1 \leq n \leq 2 \times 10^5$，$1 \leq k \leq 10^5$）。
第二行 $n$ 个整数 $a_1, a_2, \ldots, a_n$（$1 \leq a_i \leq 10^8$）。

**输出格式**

输出满足条件的子段数量。

**样例输入**

```
1 1
1
4 2
6 3 8 1
```

**样例输出**

```
1
2
```
