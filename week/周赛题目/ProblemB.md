# Problem B. 移动括号

Polycarp 有一个长度为 $n$ 的括号序列 $s$，其中 $n$ 是偶数，$s$ 由 $n/2$ 个左括号 '(' 和 $n/2$ 个右括号 ')' 组成。

每次操作，Polycarp 可以选择一个括号，将其移动到字符串的开头或结尾。

请问最少需要多少次操作，才能将 $s$ 变成一个合法的括号序列？

合法括号序列定义如下：
- "()" 是合法的；
- 如果 $A$ 是合法的，则 "(" + $A$ + ")" 也是合法的；
- 如果 $A$ 和 $B$ 是合法的，则 $A$ + $B$ 也是合法的。

\InputFile
第一行一个整数 $t$（$1 \le t \le 2000$），表示测试用例组数。

每组测试用例包含两行：
- 第一行一个偶数 $n$（$2 \le n \le 50$）。
- 第二行一个长度为 $n$ 的字符串 $s$，由 $n/2$ 个 '(' 和 $n/2$ 个 ')' 组成。

\OutputFile
每组测试输出一行，输出最少操作次数。

\Examples
```
4
2
)(
4
()()
8
())()()(
10
)))((((())
```

```
1
0
2
3
```

\Note
第一组：可以通过将第一个括号移动到末尾使序列合法。
第二组：序列已经是合法的，因此不需要任何操作。
