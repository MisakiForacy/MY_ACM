### 具体问题
> 从一个大小为1e5的数组中取出若干个数，使其异或和为x，或报告不存在

- **分析**
    如果直接使用包含位置信息的线性基插入或查找，会插入n次，每次插入复杂度为$O(64 * n)$，总复杂度变成$O(64 * n^{2})$，很显然不可接受

- **解决方法**
    可用线性基+回溯
    ```cpp
    int cur = 1;
    vector<LL> base(64, 0), mask(64, 0), use(64, 0);
    auto insert = [&](LL x, int idx) -> void {
        LL p = 1LL << cur;
        for (int j = 63;j >= 0;j --) {
            if (x >> j & 1) {
                if (!base[j]) {
                    base[j] = x;
                    mask[j] = p;
                    use[cur ++] = idx;
                    break;
                }
                x ^= base[j];
                p ^= mask[j];
            }
        }
    };
    ```
