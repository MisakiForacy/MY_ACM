# 带权并查集(路径压缩)

### 思想

> 找到路径上的所有结点，利用**递归***记忆化*将其归为根结点

```cpp
void init(int n){   // 初始化
    for (int i=0;i<=n;i++){
        Set[i] = i;
    }
}
// > 查找时间复杂度：O(nlogn)
int findx(int x){      // 查询祖宗结点
    if (Set[x] != x){
        int fa = Set[x];
        Set[x] = findx(Set[x]);
        siz[x] += siz[fa];
    }
    return Set[x];
}
// > 归并时间复杂度：O(1)
void merge(int a,int b) {       // 归并
    int b = findx(b);
    int a = findx(a);
    if (a > b) swap(a,b);
    if (a != b){
        Set[b] = a;
        siz[a] += siz[b];
    }
}
```
