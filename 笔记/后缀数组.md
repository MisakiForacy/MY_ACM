### 后缀数组SA

* 给一个字符串s
* $sa_i$ 表示将$s$的所有后缀排序后第$i$小的后缀的编号
* $rk_i$ 表示后缀$i$的排名

### 如何计算后缀数组

* 倍增求后缀数组，$O(n \log^2(n))$
```cpp
void SA(string s, vector<int>&sa, vector<int>&rk) {
    int n = s.size();
    s = ' ' + s;
    sa.resize(n + 1), rk.resize(2 * n + 1);
    for (int i = 1;i <= n;i ++) sa[i] = i, rk[i] = s[i];
    for (int j = 1;j < n;j <<= 1) {
        sort(sa.begin() + 1, sa.end(), [&](int x, int y) {
            return rk[x] == rk[y] ? rk[x + j] < rk[y + j] : rk[x] < rk[y];
        });
        vector<int> tp = rk;
        for (int p = 0, i = 1;i <= n;i ++) {
            if (tp[sa[i]] == tp[sa[i - 1]] && tp[sa[i] + j] == tp[sa[i - 1] + j]) {
                rk[sa[i]] = p;
            } else {
                rk[sa[i]] = ++ p;
            }
        }
    }
}
```

#### 求可重叠最长子串

* 相邻两项sa的lcp的最大值

#### 不可重叠最长子串

* 二分答案求解
